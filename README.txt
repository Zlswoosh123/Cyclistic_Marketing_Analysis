This project was created as an analysis of biking behavior between member and casual riders with the goal of recommendations to a marketing team on how to convert members.
This project was done in R and guided by Google's Data Analytics Capstone prompts. All analysis and work is by the individual (Zac).
Dataset links and additional info may be found in the project.
